# Six months of JavaScript - wrap-up

I started this repo in February 2018 knowing almost nothing, and now it is
the end of July. Looking back at the months:

- **Feb** - fundamentals: variables, types, functions, loops, arrays, objects.
- **Mar** - arrays & objects in depth: map/filter/reduce, destructuring, spread.
- **Apr** - functions in depth: closures, `this`, currying, composition.
- **May** - async: callbacks, Promises, async/await, fetch.
- **Jun** - OOP: prototypes, classes, inheritance, modules.
- **Jul** - tooling & projects: npm, Babel, Webpack, ESLint, Prettier, Jest.

## What clicked

- Functions are values you can pass around, return, and remember (closures).
- "Don't mutate - make a new copy" with spread, which keeps data predictable.
- async/await turned scary async code into readable, top-to-bottom code.
- Classes are just nice syntax over prototypes.
- Tests give me the confidence to change code without fear.

## What I want to keep practicing

- Reading other people's code and recognising these patterns.
- Writing tests FIRST sometimes, not just after.
- Keeping logic separate from input/output (it made testing so much easier).

## Next: React

I am genuinely excited to start React next month. The plan is components,
props and state, then lifecycle methods, then a few small apps. Onwards!
